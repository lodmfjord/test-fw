# Libs

Place shared packages in this folder.

## Naming

- `libs/<package-name>/`

## Minimum package files when you start one later

- `package.json`
- `tsconfig.json`
- `src/`
- `src/index.ts`
- `src/index.test.ts`

## Guardrails

- At most one exported function per file.
- At most 300 lines per file.
- Tests are required before implementation changes.
